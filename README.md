Hi 👋 My name is Artem
======================

Full Stack web developer
------------------------

I've been programming since I was about 13 years old, I started with creating bots for VKontakte, then I didn't even try to learn courses or read the material, I just googled every thing I didn't understand, after a while I was already confident in JavaScript, started projects on Node.Js and since about 16 years old I've been developing bots in Telegram. I could confidently write the functionality I needed, I didn't need Google so much anymore. Now I'm studying Vue.js and React.js, I haven't decided what I like more, but I can handle small projects in these languages without any problems. Since I work alone I don't separate Front End and Back End, I work on both parts, but I think I'm more satisfied with the Back End part.
